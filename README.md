# Simple Two-Page Website

This is a beginner-friendly web development project created using HTML and basic CSS. The website contains two pages with simple navigation to demonstrate the basic structure of a multi-page website.

## Features
- Built using pure HTML and CSS
- Simple and clean layout
- Two web pages (Home and Contact)
- Basic navigation between pages
- Beginner-friendly project structure

## Technologies Used
- HTML5
- CSS3

## Project Structure
index.html – Home page of the website  
contact.html – Contact page  
style.css – CSS styling for the website

## Purpose of the Project
The purpose of this project is to practice basic web development concepts such as HTML page structure, CSS styling, and linking multiple pages together.

## Future Improvements
- Add JavaScript for interactivity
- Improve UI design
- Add form validation to the contact page
- Make the website fully responsive
